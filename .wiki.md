# Project Summary
The project is focused on developing a food delivery application that allows users to order from various restaurants and grocery stores in Kuwait. It enhances the convenience of food ordering by providing a user-friendly interface and a wide selection of cuisines, including burgers, pasta, and chicken dishes. The application aims to streamline the ordering process and improve customer satisfaction.

# Project Module Description
The application consists of the following functional modules:
- **Menu Display**: Shows a categorized list of food items available for order.
- **Order Management**: Handles the user's cart, including item selection and checkout processes.
- **User Account Management**: Allows users to create accounts, save addresses, and manage payment methods.
- **Promotions and Discounts**: Displays current offers and allows users to apply discount codes.

# Usage
To set up the project:
1. Clone the repository.
2. Navigate to the project directory.
3. Install dependencies using a package manager like npm or yarn.
4. Build the project using the build command.
5. Run the application locally.
